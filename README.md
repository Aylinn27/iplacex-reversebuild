# ReverseBuildApp

Sistema web para la documentación de procesos de desarme técnico y generación automática de guías de ensamblaje inverso.

## Tecnologías utilizadas
- Node.js
- Express
- MongoDB Atlas
- HTML / CSS / JavaScript
- JWT para autenticación

## Estructura del proyecto
- frontend/: Interfaz de usuario
- backend/: API REST y lógica de negocio
- docs/: Documentación y evidencias

## Ejecución local
1. Clonar repositorio
2. Ejecutar `npm install` en /backend
3. Configurar archivo `.env` con variables de entorno
4. Ejecutar `npm run dev`
5. Abrir `frontend/index.html`

## Accesos de prueba
- Usuario técnico (demo)
- Credenciales creadas desde la interfaz de login
